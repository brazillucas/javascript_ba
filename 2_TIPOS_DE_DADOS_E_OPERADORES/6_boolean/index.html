Booleans

  • Serve para guardar um valor de uma comparação, por exemplo:
    • console.log(“1” == 1);
    • console.log(“1” === 1);
    • console.log(“1” != 1);
    • console.log(“1” !== 1);
    • console.log(true);
    • console.log(false);
    • console.log(true && false);
    • console.log("1" == 1);
    • console.log("1" === 1);
    • console.log("1" != 1);
    • console.log("1" !== 1);
    • console.log("1" > 1);
    • console.log("1" < 1);
    • console.log("1" >= 1);
    • console.log("1" <= 1);
    • console.log("1" && 1);
    • console.log("1" || 1);
    • console.log(!1);
    • console.log(!true);
    • console.log(!false);
    • console.log(!(1 > 2));
    • console.log(!(1 < 2));
    • console.log(!(1 >= 2));
    • console.log(!(1 <= 2));
    • console.log(!(1 && 2));
    • console.log(!(1 || 2));
    • console.log(!(true && false));
    • console.log(!(true || false));
  • Os únicos valores possíveis são:
    • True;
    • False;

Confira o console para mais exemplos.

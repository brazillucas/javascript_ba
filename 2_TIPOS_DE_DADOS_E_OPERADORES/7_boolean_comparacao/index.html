Booleans: comparações

  • Maior que: >
  • Menor que: <
  • Maior ou igual: >=
  • Menor ou igual: <=
  • Igual: ==
  • Diferente: !=
  • Idêntico: ===
  • Não idêntico: !==
  • Verdadeiro: true
  • Falso: false
